# Video Trending App 📺

![Video Trending Banner](https://images.unsplash.com/photo-1611162618071-b39a2ec055fb?w=800)

Discover and track trending videos across platforms.

## ✨ Features

- 🔥 Trending videos
- 📊 View counts
- 📈 Trending charts
- 🔔 Notifications
- 🎯 Category filters

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/antono4/video-trending-app.git

# Open in browser
cd video-trending-app
open index.html
```

## 📁 Project Structure

```
video-trending-app/
├── index.html      # Main page
├── css/            # Stylesheets
├── js/             # JavaScript files
└── assets/         # Static assets
```
